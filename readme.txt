Integrates Post Promoter Pro with the freely available Subtitles plugin

== Description ==

Post Promoter Pro - Subtitles adds support for the freely available Subtitles plugin by Phillip Arthur Moore (https://wordpress.org/plugins/subtitles/)

It adds the <pre>{post_subtitle}</pre> replacement to allow Post Promoter Pro to include it in your social media posts.

== Installation ==

= Using the WordPress Dashboard =
1. Navigate to *Plugins -> Add New* from within the WordPress Dashboard.
2. Search for `ppp-subtitles`.
3. Click **Install Now** on *Post Promoter Pro - Subtitles*.
4. Activate the plugin.
